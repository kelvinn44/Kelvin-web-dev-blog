Week 7 Web Dev Blog

DOM, Event Handling & Validation, and JSON Local Storage

  • I've learnt about DOM (Document Object Model).

  • I've learnt about DOM Tree, how to access elements (document.) using the getElementById(); , getElementsByClassName(); , getElementsByTagName(); , querySelector(); , querySelectorAll() , etc.

  • I've learnt about event handling like click, element.addEventListener , and Event Object.

  • I've also learnt about validation for forms etc.

  • Lastly, I've learnt about JSON Local Storage, where they are stored, their format, how to view and retrive them, why they are used, etc.
