Javascript Basic

  • I've learnt that version ES5 of JavaScript has the best compatibility with modern web browsers.

  • I've learnt that if I were to plan on using JavaScript for backend development, the best choice would be to use node.js

  • I've also learnt that object in JavaScript is an unordered set of key-value pairs called properties.

  • Lastly, I've learnt what is a strict equality operator === and the strict inequality operator !== . The difference between the strict equality operator === in JavaScript compared to the equality operator == is that the === operator compares values and data types, while the == operator compares only values.

  • Overall, after learning more about JavaScript, I feel that it is somewhat similar to Python. With the way on naming a variable, creating, manipulating and accessing a arrays, and using strings etc.
